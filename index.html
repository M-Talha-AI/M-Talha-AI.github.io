Wealth Genie — Financial Advisory System

Timeline: January 2025 - Present

Client: WealthNX

Description:

Wealth Genie is a sophisticated financial advisory system designed to provide personalized financial guidance to users of the WealthNX mobile platform. The system combines the power of Knowledge Graphs with Retrieval-Augmented Generation to deliver accurate, contextually relevant financial advice.

Key Features:

  • Multi-agent architecture that simulates a team of financial experts with specialized roles
  • Knowledge Graph integration for complex financial relationship modeling
  • Privacy-preserving design utilizing local compute resources for sensitive financial data
  • Hybrid cloud architecture with Azure for scalability and compliance
  • Personalized financial planning based on user goals and risk tolerance

Technologies Used:

LangChain, LangGraph, Neo4j, Azure Cloud, PyTorch, Ollama Transformers, ChromaDB

Outcome:

The system is currently in final testing and scheduled for deployment on the WealthNX mobile platform (Android and iOS) in Q3 2025. Early testing shows 35% higher user satisfaction compared to previous rule-based financial advisory tools.

Zero Violation — Traffic Monitoring System

Timeline: 2023 - 2024 (Final Year Project)

Client: Self-initiated project

Description:

Zero Violation is a real-time traffic monitoring system implemented on Jetson Nano that can detect and record traffic violations automatically. The system uses computer vision techniques to identify vehicles, track their movements, and detect violations such as Helmet, Seatbelt, Wrong Number-Plate, and improper lane changes.

Key Features:

  • Real-time object detection using YOLOv4 customized for edge devices
  • Custom post-processing algorithms for violation identification and classification
  • Vehicle tracking across video frames with high accuracy even in challenging lighting conditions
  • Local storage of violation events with automatic alert generation
  • Web-based dashboard for monitoring and reviewing recorded violations

Technologies Used:

YOLOv4, Darknet, OpenCV, Jetson Nano, TensorRT, Docker, Roboflow, Flask

Outcome:

The system achieved 92% accuracy in violation detection during field tests. The project received top honors in the university's project showcase and attracted interest from local traffic authorities for potential implementation in urban areas.
